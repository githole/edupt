edupt
=====

eduptはC++で書かれたパストレーシングによるシンプルでコンパクトな物理ベースレンダラです。 ソースコードはGithubで公開されており、日本語によるコメントが付けられています。  
さらに、パストレーシング・物理ベースレンダリング・eduptのソースコード、の解説スライドも公開しています。 

http://kagamin.net/hole/edupt/index.htm
